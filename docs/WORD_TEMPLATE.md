# 集會紀錄 Word 模板與列印指南

> 本點名系統提供三種輸出方式：瀏覽器直接列印、動態生成 .docx、以及 CSV 匯出後做 Mail Merge。

---

## 方法一：瀏覽器直接列印（最簡單） ⭐ 推薦

適合：快速列印、不需要存檔、即時使用

1. 在點名系統中選擇支部，按「📊 查看矩陣」
2. 或者直接選擇日期載入點名表
3. 按工具列的 **「🖨️ 列印」** 按鈕
4. 瀏覽器會打開一個排版好的列印預覽頁面
5. 按 `Ctrl+P`（Windows）或 `Cmd+P`（Mac），或按頁面上的「列印」按鈕
6. 選擇印表機，或選「另存為 PDF」

> ✅ 優點：最快、最簡單、排版已最佳化
> ❌ 缺點：無法直接存為 .docx 格式

---

## 方法二：動態生成 Word（.docx）

適合：需要存檔、交報告、需要後續編輯

1. 在點名系統中選擇支部，按「📝 下載 Word」
2. 瀏覽器會自動下載 `出席紀錄_支部_日期.docx`
3. 用 Microsoft Word 或 Google Docs 開啟
4. 可能會看到 Word 提示「是否需要修復」，這是因為檔案是動態生成的，**按「是」即可正常開啟**

> ✅ 優點：直接產生 .docx，可後續編輯
> ❌ 缺點：舊版 Word 可能會提示修復；大型表格可能較慢

---

## 方法三：CSV 匯出 + Mail Merge（最靈活）

適合：需要正式排版（加入旅團徽章、特定格式、多頁表頭）

### 3.1 匯出 CSV

1. 在點名系統中選擇支部，按「📄 匯出 CSV」
2. 下載 `出席紀錄_支部_日期.csv`
3. 用 Excel 或 Google Sheets 開啟確認內容正確

### 3.2 在 Microsoft Word 建立 Mail Merge 範本

**步驟一：設計表格範本**

開啟 Word，設計你想要的集會紀錄表格式。例如：

```
╔══════════════════════════════════════════════════════════╗
║           第 82 旅 集會出席紀錄                          ║
║  支部：童軍團        集會日期：________________         ║
╠══════════════════════════════════════════════════════════╣
║ 姓名：«姓名»  小隊：«小隊»  YMIS：«YMIS號»              ║
╠══════════════════════════════════════════════════════════╣
║                                                          ║
║  日期        狀態                                        ║
║  «2026-06-01»  «2026-06-01»                            ║
║  «2026-06-08»  «2026-06-08»                            ║
║  ...                                                     ║
║                                                          ║
╚══════════════════════════════════════════════════════════╝
```

**步驟二：插入合併欄位**

1. 在 Word 中，上方選單 **郵件** → **開始郵件合併**
2. 選擇文件類型：
   - 如果要「一人一格」的表格：選 **「目錄」**（Directory）
   - 如果要「一人一份」的獨立文件：選 **「信件」**（Letters）
3. **選取收件者** → 使用現有清單 → 選擇你的 CSV 檔
4. 在需要插入資料的地方游標定位，然後按 **插入合併欄位**：
   - `«姓名»`
   - `«小隊»`
   - `«YMIS號»`
   - `«2026-06-01»`（日期欄位名稱每次可能不同，需手動對應）
   - `«2026-06-08»`
   - ...

**步驟三：預覽並完成**

1. 按 **預覽結果**（預覽結果）查看效果
2. 確認無誤後，按 **完成與合併** → 編輯個別文件
3. 產生完整的紀錄表

> 💡 **小技巧**：日期欄位每次匯出可能名稱不同。建議建立範本時用固定的前幾個日期（如日期1~日期10），或每次重新對應欄位。另一個方式是：先累積一定數量的集會（如10次），再匯出CSV，這樣日期欄位比較穩定。

---

## 欄位對照表（CSV 結構）

點名系統匯出的 CSV 第一列為表頭：

| 欄位 | 說明 | 範例 |
|------|------|------|
| `姓名` | 成員姓名 | 陳大文 |
| `支部` | 所屬支部 | 童軍團 |
| `小隊` | 所屬小隊/六 | TIGER |
| `YMIS號` | 成員 YMIS 編號 | 0123456789 |
| `2026-06-01` | 該日期的出席狀態 | P / A / L / E / S |
| `2026-06-08` | 下一個日期... | P / A / L / E / S |

## 靜態 Word 範本檔（.docx）

如果你需要一份預先設計好的 `.docx` 範本檔（含 `MERGEFIELD` 域），可以：

1. 先用上述方法三製作一份範本
2. 存檔為 `出席紀錄範本.docx`
3. 日後每次：
   - 匯出新 CSV
   - 開啟範本
   - 郵件 → 選取收件者 → 重新連接新 CSV
   - 完成合併

> 本系統未提供靜態範本檔（因日期欄位動態變化），但你可以按照上述方法自行建立一次，之後重複使用。

## 比較表

| 方法 | 難度 | 適合情境 | 輸出格式 |
|------|------|---------|---------|
| 瀏覽器直接列印 | ⭐ 最簡單 | 快速列印、即時使用 | 紙張 / PDF |
| 動態生成 .docx | ⭐⭐ 簡單 | 存檔、交報告、後續編輯 | .docx |
| CSV + Mail Merge | ⭐⭐⭐ 中等 | 正式排版、多頁格式、徽章 | .docx / PDF |

> 建議：日常點名用「瀏覽器列印」或「動態 .docx」；學年終報告或區審查用「Mail Merge」做正式排版。
